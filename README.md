# Cyberpunk Tic-Tac-Toe

A futuristic Tic-Tac-Toe web app where you play as **X** against an AI **O** opponent in a best-of-three match.

## Run locally

1. Open the project folder in your terminal.
2. Start a local server:

```bash
npx serve .
```

3. Open the local URL shown in terminal.

## Gameplay

- You play first as **X**.
- The AI plays as **O**.
- Match winner is decided by best of 3 rounds.
- Use **Next Round** after each round, and **Play Again** after match end.

## Troubleshooting

If you still see an old version of the app:

1. Update your local branch:

```bash
git pull origin main
```

2. Restart the local server:

```bash
npx serve .
```

3. Refresh the browser tab.

If `npx serve` is already running and terminal seems stuck, press `Ctrl + C` first.
